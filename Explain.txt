Mikufy v2.11-nova - Fedora43、ArchLinux、Debian13 依赖包列表


必需依赖：


Fedora43(或其他较新的RHEL系)
sudo dnf install webkitgtk6.0-devel gtk4-devel file-devel json-devel gcc-c++ pkg-config glib2-devel file-libs

*

ArchLinux(或其他较新的ArchLinux系)
sudo pacman -S pkgconf base-devel webkitgtk6.0 gtk4 glib2-devel file nlohmann-json

*

Debian13(或其他较新的Debian系)
sudo apt install pkg-config  libwebkitgtk-6.0-dev file libgtk-4-dev nlohmann-json3-dev g++ libglib2.0-dev libmagic-dev
